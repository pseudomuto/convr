# convr

[![CI Status][ci-svg]][ci-url]

A little CLI tool for converting between units.

## Usage

```bash
A simple little program to convert values between units

Usage: conv [OPTIONS] <FROM> <TO_UNIT>

Arguments:
  <FROM>
  <TO_UNIT>

Options:
  -u, --units
  -h, --help     Print help
  -V, --version  Print version
```
